 # 开发
 
 ## 准备工作
 
 ```
    1.确保node和npm的安装，均是最新版本。Node：v6.9.1；npm：3.10.8。
    2.确保MongoDB的安装，MongoDB的安装及配置可以参考网址：http://blog.csdn.net/lvkelly/article/details/54618752
 ```

## 安装依赖

```
    1.npm安装
    npm install
    
    2.cnpm安装
    cnpm淘宝镜像安装：npm install –g cnpm –registry=https://registry.npm.taobao.org
    cnpm install
```
## 启动命令

```
    npm start
```

## 打包命令

```
    npm build
```

## 访问地址

```
    http://localhost：8900
```
